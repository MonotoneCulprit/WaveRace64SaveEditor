Wave Race 64 - On Cart Save Hex Offsets (EEPROM/.eep) 

0x0-0x1 - This seems to always stay as 54 45 (TE). Header of some kind.

0x2-0x3 - CHECKSUM
This Checksum is calculated by summing all succeeding Bytes (0x4-0x1FF).
For a blank save this is C8 63

0x4-0x7 - FF Spaces

0x8-0xC - Settings and Unlock Data?
Default is 01 00 00 00 00
100% Save shows 06 06 07 07 0F

0xD-0xF - FF Spaces

CUSTOM NAME SETTINGS

0x10-0x18 - R.Hayami Name
0x1A-0x22 - D.Mariner Name
0x24-0x2C - A.Stewart Name
0x2E-0x36 - M.Jeter Name
(Only overwrites length of custom name i.e. Custom name "TEST" Data will say TEST MI(End of R.Haya(MI) still present)
(Also true in reverse. Switching back to default only replaces the length of the default name)
Space 00 is inserted after Custom or Default name to denote end of name.

CUSTOM CRAFT SETTINGS

0x38-0x3A - Hayami Custom Craft Settings
0x3B-0x3D - Mariner Custom Craft Settings
0x3E-0x40 - Stewart Custom Craft Settings
0x41-0x43 - Jeter Custom Craft Settings

3 Bytes - Handling, Engine, Grip
Default is 05 05 05.
Each pair denotes position of Dash from 00 to 0B (0-11).

DEFAULT CRAFT SETTINGS???

0x44-0x4F - Every Byte is set to 05. Same length as Craft Settings data.

CONDITIONS

0x50-0x52 - Normal Conditions
0x53-0x55 - Hard Conditions
0x56-0x58 - Expert Conditions

3 Bytes - Wave Conditions, Number of Misses, Number of Laps
Default is 00 05 03
Wave Conditions - 00 is Normal, 01 is Calm, 02 is Wild, 03 is Random
Misses and Laps are stored as raw values (05 for 5 Misses, 03 for 3 Laps).
